# CONTRIBUTING

## New post

1. Create a new post in the `src/posts` directory 
2. Add the front matter, write the post and add any images to `src/images`

    ```md
    ---
    date: <date>
    name: <title>
    icon: <icon>
    public: True
    categories:
      - <category>
    ---
    ```

3. Preview the site by running `npm run start` and proceed if everything looks good
4. Run `npm run publish` to build the site
5. Push the changes to the remote repository
6. Marvel at your new post
